# POACHING DATA 

## Setup

1. Create Virtual Environment

```bash 
python -m venv dev 
python3 -m venv dev
```

2. Activate Virtual Environment

```bash
dev\Scripts\activate
```

or 

```bash
source dev\bin\activate 
```

3. Install  packages

```bash
pip install -r requirements.txt
```


4. Run 

```bash
python main.py
```
